# Design System - Orango

## Design Principles

- **Vibrant & Energetic**: Use bold colors and playful elements
- **Clean & Modern**: Maintain clean layouts and modern aesthetics
- **Consistent**: Ensure visual consistency across all components
- **Playful yet Professional**: Balance playfulness with usability

---

## Color System

### Primary Colors
- **Brand Orange**: Main brand accent color
- **Dark Orange**: Deeper tone for depth and contrast
- **White**: Primary background and text color on dark
- **Light Cream**: Soft warm background alternative

### Color Combinations
- Orange on White/Cream
- White on Orange/Dark Orange
- Multiple color variation versions available

### Color Usage Guidelines
- Use Brand Orange as primary accent
- White for clean backgrounds
- Maintain contrast ratios for accessibility
- Use color variations consistently

---

## Typography System

### Type Hierarchy
- **Display Text**: Large, bold for maximum impact
- **Headers**: Medium-large for section titles
- **Body Text**: Readable sizes for content
- **Branding Lines**: Playful, expressive typography

### Typography Styles
- **Logo Text**: Customized typography for "开个橘子" or "ORANGO"
- **Headers**: Bold weights for emphasis
- **Body**: Clean, readable sans-serif
- **Accent**: Playful variations for branding elements

### Font Weights
- Bold for headers and logo
- Regular for body text
- Varied weights for hierarchy and emphasis

---

## Spacing & Layout

### Spacing Scale
- Consistent spacing system for all elements
- Maintain rhythm and visual harmony
- Use multiples of base spacing unit

### Layout Principles
- **Balanced Compositions**: Create visual balance in all layouts
- **Clear Hierarchy**: Establish clear information hierarchy
- **Consistent Alignment**: Maintain alignment consistency
- **Adequate Whitespace**: Use whitespace strategically

### Grid System
- Flexible grid for responsive layouts
- Consistent column structure
- Proper gutters and margins

---

## Components

### Buttons
- Use 20px border radius as standard
- High contrast between background and text
- Clear hover and active states

### Cards & Containers
- 20px border radius for rounded containers
- 200px border radius for special circular elements
- Clean backgrounds with proper padding
- Consistent padding and spacing

### Frames
- Use proper padding and spacing
- Consistent corner radii
- Clear visual separation

---

## Border Radius

### Standard Radius
- **20px**: Default for buttons, cards, and containers
- **200px**: For special circular or highly rounded elements
- **0px**: For sharp corners where needed

### Usage
- Use 20px for most UI elements
- Reserve 200px for special branding elements
- Be consistent with radius usage across similar components

---

## Imagery & Graphics

### Image Format
- Prefer SVG for scalability and crispness
- Maintain consistent image style
- Coordinate with brand color palette

### Image Treatment
- Use SVG graphics for icons and logos
- Maintain aspect ratios
- Ensure images load efficiently

### Graphic Elements
- Clean vector graphics
- Consistent stroke weights
- Coordinate with brand colors

---

## Design Do's & Don'ts

### Do's
- Use consistent border radii (20px standard)
- Maintain color consistency
- Establish clear typography hierarchy
- Use proper spacing and alignment
- Keep designs clean and modern

### Don'ts
- Don't mix border radii inconsistently
- Don't use unapproved color combinations
- Don't ignore typography hierarchy
- Don't overcrowd layouts
- Don't use inconsistent spacing
